# Lore

This directory serves as the foundational knowledge base for the `mystic_codex`. It is structured to help agents contextualize and analyze the corpus across different traditions.

## Categories

- `beings_and_deities/`: Entities, gods, and spirits.
- `characters/`: Key figures, archetypes, and historical persons.
- `communities_and_teachers/`: Lineages, schools, and influential figures.
- `frameworks_and_models/`: Systems of thought, cosmologies, and maps of reality.
- `phenomenology_and_concepts/`: Subjective experiences, altered states, and theoretical concepts.
- `places_and_realms/`: Physical locations, subtle realms, and mythic geographies.
- `spells_and_practices/`: Rituals, meditations, and techniques.
- `texts_and_authors/`: Foundational texts, grimoires, and writers.
