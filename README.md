# Mystic Codex

The goal of this repository is to create a set of agent skills and tools for corpus analysis of a collection of PDFs coming from specific mystic and esoteric traditions. 

This project is designed to facilitate exploration, knowledge extraction, and cross-tradition analysis. 

## Structure
- `corpus/`: Example texts and documents for analysis.
- `skills/`: Agent-specific skills for interacting with and interpreting the corpus.
- `tools/`: Python scripts and utilities for PDF parsing and data management.
- `lore/`: The foundational knowledge base, categorized into characters, practices, realms, frameworks, and more.

## Setup
Install Anaconda from https://www.anaconda.com/download/success?reg=skipped

Please refer to `conda_setup.md` for instructions on creating the virtual environment for this project.
